Índices extremos

Os índice extremos foram criados para se mensurar eventos climáticos, principalmente os intensificados pela mudança climática. Temos 40 índices para os mais variados eventos, serão visitados poucos índices, visto que muitos deles não são aplicados para o Brasil ou são pouco usados. A totalidade pode ser vista em Climdex, ETCCDI e Anexo IPCC AR6. Sendo o primeiro mais explicativo, enquanto o segundo apresenta as formulações de forma mais clara e o último é uma mistura de ambos. Para o índice SPI foi usado a referencia Standard precipitation index Para implementações, usamos a biblioteca AgenciBr para implementações em python e o Climate Date Operator (CDO)
